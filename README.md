# AllGrownUp

AllGrownUp is a back end service running on Laravel and integrated with Shopify.

## Installation

```bash
composer install
php artisan migrate
php artisan passport:install
```

### Run queue worker
```bash
php artisan queue:work
```

## Usage

```bash
php artisan serve
```
